## 🧰 Installation

``` bash

git clone https://github.com/cppandpython/windows_bot.git

# Edit in bot_telegram.py or bot_socket.py

    TOKEN = 'YOUR TELEGRAM BOT TOKEN'

# Launch
python bot.py
```
